
Transportation Manager

A Transportation Manager desktop software. The product manages transportation services operations and maintenance. It handles invoicing, currency exchange, inventory, clients and suppliers databases, vehicles maintenance, scheduling, payroll and accounts payable. 

Requirements:
Windows xp or 7
MySQL Server 4.1
MyODBC windows drivers 3.51
BDE Admin 5.01
